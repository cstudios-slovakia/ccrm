##unified universal entry
    - unified entries is a concept which allow the crm to store any kind of datastructure
    - to create an unified entry the admin must move to the settings, and there to the tab called unified entries and create one
    - every univied entry will have a icon and a color which can be selected
    - when the entry is created it is registered in the unified entry database as a row and a new database table is spawned, where each row will be a data entry and each collumn a parameter for the selected modules
    - every unified entry can have modules attached. For the first version we will add
        - due date
        - file
        - title
    - these modules will define what data is set when an entry is created
    - when the unified entry is removed the data will remain, only an archive flag will be set in the the unified entry registry database
    - in the unified entries settings (in the system settings) there should be a possibility to turn on or off folder.
    - folder will group unified entries together and can have the same modules as the unified entry
